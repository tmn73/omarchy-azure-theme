# Azure - Sky-blue theme for Omarchy

A carefully crafted theme that brings the serene aesthetics of clear skies to your desktop environment. Azure features a cohesive sky-blue color palette across all system components, creating a calm and productive workspace. Built specifically for Omarchy, this theme provides consistent styling for your window manager, status bar, applications, and more.
